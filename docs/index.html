Smart Shoe System for Obstacle Detection and Guidance

Abstract

This research explores the design and implementation of a smart shoe system to aid visually impaired individuals by detecting obstacles using microcontrollers and sensors. The system integrates ultrasonic and infrared sensors to provide real-time feedback via Bluetooth earpieces.

Introduction

For visually challenged people, safely navigating dynamic surroundings is rather difficult. Although they provide simple tactile input, traditional mobility aids like canes are not very good at identifying objects that are not exactly in front of the user or far away. This work presents a smart shoe system combining microcontroller technology, sensor integration, and wireless connection to deliver real-time input for obstacle identification and navigation, thereby addressing these constraints.

Along with ultrasonic and infrared (IR) sensors, the suggested smart shoe uses reasonably priced and easily available technologies such as Arduino and Raspberry Pi to provide a complete solution for safe navigation. This work aims to create a wearable device using Bluetooth earphones that improves mobility, lowers accidents, and offers an easy user experience by means of auditory signals.

Especially in the field of obstacle detection, wearable assistive devices have developed dramatically recently. Current options include smart glasses, wearable belts with sensors that offer tactile or aural input, and electronic canes. In complicated settings, these tools can lack comfort, simplicity of use, or accuracy, however.

Research on low-cost, adaptable, and efficient prototypes for assistive devices has extensively embraced microcontrollers like Arduino and Raspberry Pi. For instance, ultrasonic sensors and vibration feedback systems have been used in designs for smart shoes with simple obstacle-detecting powers. These designs can lack directionality and the capacity to differentiate between different kinds of barriers, including fixed and moving items, or ground-level dangers like pits.

By adding cutting-edge technologies such as multi-directional obstacle detection, real-time Bluetooth connection, and a decision-making algorithm to provide the user clear, practical auditory instructions, this study seeks to expand upon already existing concepts.
